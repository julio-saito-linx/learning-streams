# stream-adventure
https://github.com/substack/stream-adventure

## install stream dependencies
- through
- concat-stream
- duplexer
- trumpet
- request
- split
- tar
- websocket-stream
- stream-combiner
```
npm i
```


## install class
```
git clone git@github.com:substack/stream-adventure.git
cd stream-adventure
npm i
```

## start
```
./sa
```

## verify
```
./sa verify program.js
```
